# KEEP - Vivid Note 📝

A beautiful note-taking Progressive Web App built with vanilla JavaScript and Supabase.

## Features
- 🔐 User authentication (sign up / login)
- 📝 Create, edit, delete notes
- 📌 Pin important notes
- 🏷️ Custom labels and categories
- ✅ Checklists with interactive checkboxes
- 🎨 8 color themes for notes
- 🔍 Real-time search
- 📋 Duplicate notes

## Tech Stack
- **Frontend:** Vanilla JavaScript, Tailwind CSS
- **Backend:** Supabase (Auth + PostgreSQL)
- **Design:** Stitch "Vivid Note" design system
- **Font:** Spline Sans + Material Symbols

## Setup
1. Clone this repository
2. Set up a Supabase project
3. Update `supabase-client.js` with your Supabase URL and anon key
4. Run `setup.sql` in Supabase SQL Editor
5. Enable Email provider in Supabase Auth settings
6. Serve with any static file server
